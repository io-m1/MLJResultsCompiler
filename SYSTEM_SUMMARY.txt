WHAT GETS PROCESSED
================================================================================

INPUT FILES (5 monthly test files):
- TEST_1_*.xlsx - Initial test results (any # of questions)
- TEST_2_*.xlsx - Follow-up test results (any # of questions)
- TEST_3_*.xlsx - Midterm exam results (any # of questions)
- TEST_4_*.xlsx - Practical exam results (any # of questions)
- TEST_5_*.xlsx - Final/Ultrasonography test results (any # of questions)

OUTPUT FILE (single unified result sheet):
- OBS_[MONTH_YEAR]_RESULT_SHEET.xlsx
  Contains:
  • S/N (serial number)
  • NAMES (participant name)
  • EMAIL (participant email)
  • TEST 1-5 (individual test scores as percentages)
  • GRP DISCUSSION (fixed at 0.8 points)
  • TOTAL MARK (sum formula)
  • SCORE (weighted calculation formula)
  • STATUS (PASS if >=50%, FAIL if <50%)

ADDITIONAL OUTPUTS:
- collation_log_*.json - Error tracking and audit trail
- validation_report_*.json - Data quality metrics
- execution_log_*.json - Process history and timestamps
